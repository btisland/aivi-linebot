## Technical Details

### 目錄結構
```
aivi-linebot/
├── src/
│   ├── scrapers/        # 爬蟲模組
│   │   └── __init__.py
│   ├── handlers/        # 指令處理器
│   │   └── __init__.py
│   ├── utils/           # 工具函式
│   │   └── __init__.py
│   └── app.py           # Webhook 服務入口
├── tests/               # 測試檔案
│   ├── test_scrapers/
│   └── test_handlers/
├── .env.example         # 環境變數範本
├── pyproject.toml       # 套件設定
└── README.md
```

### 套件相依
在 `pyproject.toml` 的 `dependencies` 加入：
- `line-bot-sdk>=3.0.0`
- `httpx>=0.27.0`
- `selectolax>=0.3.0`
- `flask>=3.0.0`

在 `optional-dependencies[dev]` 加入：
- `pytest>=7.0.0`
- `pytest-cov>=4.0.0`
- `pytest-asyncio>=0.21.0`
- `pytest-mock>=3.10.0`

### 環境變數
建立 `.env.example`：
```bash
LINE_CHANNEL_SECRET=your_channel_secret_here
LINE_CHANNEL_ACCESS_TOKEN=your_access_token_here
```
